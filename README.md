# GraphML-to-CSV
Convert a family tree layout yEd GraphML file to CSV format.

https://docs.google.com/spreadsheets/d/1myXZxHrm9QIRGNq-uBq44zx7zjkzZFR6-tgZx_jfNR8/copy

2. download as excel

3. open in Yed

4. save as graphML

5. convert graphML to tab separated
